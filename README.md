# SunTrack Web Platform ☀️

This repository contains the source code for the official website of **SunTrack**, a startup focused on renewable energy solutions. Although the project did not reach the public launch phase, the platform remains a testament to modern web development standards and clean UI/UX design.

## 🚀 Key Features
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **Performance Focused:** High-speed loading through optimized assets and clean logic.
- **Interactive UI:** Engaging user experience with smooth transitions and modern aesthetics.
- **SEO Optimized:** Built with search engine visibility in mind.

## 🛠️ Tech Stack
- **Frontend:** Next.js , tailwindcss 
- **Architecture:** Component-based approach
- **Deployment:** [Например: Vercel / Netlify]

## 📝 Project Context
I was responsible for the entire frontend development lifecycle. My goal was to create a visual identity for SunTrack that communicated trust and innovation in the green energy sector.
